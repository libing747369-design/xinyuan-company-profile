# 馨源公司简介网页 COS 上传说明

本目录是可直接上传腾讯云 COS 的静态网站发布包。

## 上传前确认

- 入口文件：`index.html`
- 图片与视频目录：`images/`
- 保留原入口：`company-profile.html`
- 当前包体积约 260MB，其中企业宣传片约 89MB。

## 腾讯云 COS 推荐配置

1. 进入腾讯云控制台，打开“对象存储 COS”。
2. 新建存储桶：
   - 地域：上海或南京
   - 访问权限：私有读写创建，后续在网站托管时改为公有读私有写
   - 存储类型：标准存储
   - 计费：先用按量计费
3. 进入存储桶后，上传本目录内全部内容。
   - 注意：上传的是本目录里面的文件和 `images` 文件夹，不是把整个外层文件夹再套一层上传。
4. 设置访问权限：
   - 权限管理 -> 存储桶访问权限 -> 公有读私有写
5. 开启静态网站：
   - 基础配置 -> 静态网站 -> 开启
   - 索引文档：`index.html`
   - 错误文档：可填 `index.html`
6. 保存后复制“静态网站访问节点”。

## 验收方法

打开静态网站访问节点，确认：

- 首页能直接打开，不需要手动输入 `company-profile.html`
- 首页厂区图、产品图、客户 logo、车型项目图正常显示
- 企业宣传视频能播放
- 手机浏览时导航、首页标题和产品卡片不重叠

## 后续建议

- 如果客户访问人数少，先不要开 CDN。
- 如果访问量上来，再购买 10GB 或 50GB 外网下行流量包。
- 如果要正式长期外发，建议绑定公司自有域名。腾讯云官方提示，2024 年后新建存储桶不建议长期只依赖默认域名做正式访问。
